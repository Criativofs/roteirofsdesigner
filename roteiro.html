Roteiro com IA
CRIAÇÃO DO
SCRIPT
Gancho → Problema → Prova → CTA. Simples e eficaz.
Estrutura Universal
0–3 SEG
🪝 Gancho (Hook)
Para o scroll. Pergunta, afirmação chocante ou cena visual impactante.
3–20 SEG
🔥 Problema + Solução
Mostre a dor. Apresente o produto como resposta. Fale em benefícios, não features.
20–28 SEG
✅ Prova Social
Depoimento, número, antes/depois ou demonstração prática.
28–30 SEG
📣 CTA
Uma ação clara. "Acesse o link", "Compre agora", "Salva esse vídeo".
Prompt Base para Script
Cole no ChatGPT / Claude
Crie um script para vídeo de produto: Produto: [nome e descrição breve] Público-alvo: [quem é e qual a dor principal] Plataforma: [Reels / TikTok / YouTube Shorts] Duração: [15s / 30s / 60s] Tom: [luxury / divertido / educativo / urgência] CTA: [o que deve fazer no final] Estrutura: gancho (0-3s), problema+solução (3-20s), prova social (20-28s), CTA (28-30s). Inclua: narração off, texto na tela e descrição visual.

Publicação & Análise
DISTRIBUIÇÃO
& RESULTADOS
Publique estrategicamente, teste e analise para escalar.
Prompt — Legenda Otimizada
Cole no Claude / ChatGPT
Crie 3 versões de legenda para Instagram: Produto: [nome] Objetivo: [venda / awareness / engajamento] Tom: [divertido / premium / educativo] CTA: [link na bio / arrasta / comenta] Versão A: curta (até 3 linhas) Versão B: storytelling (5–6 linhas) Versão C: com pergunta de engajamento Inclua: emojis estratégicos e 3–5 hashtags.
Estratégia A/B
SEMANA 1
Publique 3 variações de gancho
Mesmo produto, aberturas diferentes. Identifique qual retém mais nos primeiros 3s.
SEMANA 2
Teste estilos visuais
Clean/minimalista vs. energético/colorido. Veja o que converte mais.
SEMANA 3
Escale o vencedor
Invista em tráfego pago no formato que performou. Gere variações com IA.
Métricas para Acompanhar
👁️
Retenção
% até o final. Meta: >40% nos primeiros 3s.
💬
Engajamento
Comentários, salvamentos, compartilhamentos.
🛒
Conversão
Cliques no link e vendas geradas.
💰
ROAS
Retorno sobre investimento em ads.
